# Incantation Practice

信念を書き換えるインカンテーション練習ツール。

ブラウザだけで動くシンプルな1ファイルアプリです。スマホでもPCでも使えます。

**[今すぐ試す → https://knowledge-nudge.github.io/incantation-practice/](https://knowledge-nudge.github.io/incantation-practice/)**

## Incantation とは

ラテン語 "incantare"（歌い込む）が語源。アンソニー・ロビンズが提唱する、**身体・声・感情を総動員して信念を書き換える**手法です。

頭で言い聞かせるアファメーションと異なり、繰り返し＋感情で信念レベルまで落とし込むのが特徴です。

## 2つのモード

### EB モード（Empowering Belief）

Empowering Belief（力を与える信念）をカラオケ風に表示。声に出して唱えることで身体に染み込ませます。

### LB折りモード（Limiting Belief）

Limiting Belief（制限的な信念）を5ステップで書き換えます。

1. Limiting Belief を認識する
2. **折ると決める** — 決断が全て
3. **折るのは簡単** — 難しいという思い込みを外す
4. **これはウソだ** — 反証を見つけて証明する
5. Empowering Belief へ書き換える

## 操作方法

| キー | 操作 |
|------|------|
| `Space` | 一時停止 / 再開 |
| `N` | 次のフレーズへ |
| `M` | モード切替（EB ↔ LB折り） |
| `T` | テーマ切替（ダーク ↔ ライト） |
| `F` | 全画面 |
| `R` | タイマーリセット |
| `↑` `↓` | 表示秒数の調整 |
| `H` or `?` | ヘルプ表示 |

画面右上のボタンからも全操作が可能です。

## 使い方のコツ

- 1日最低 **20分** が目安
- 声に出す。感情を込める。身体を使う
- 「私は **既に** その状態である」（Be → Do → Have）
